# Tea Cozy 
This project is part of [Codecademy's](https://www.codecademy.com/learn) *"Full Stack Engineer"* course. Its purpose is to design a responsive website regarding a fictional tea webpage.

## Description
With a given website design sheet, users must match the required design using their knowledge of the CSS property "Flexbox" in order to present a good website regardless of the device's height & width. 
All files are included in resources, alongside the website design system used to style the page.
